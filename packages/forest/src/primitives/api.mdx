# API

## Reference

### Core props

| Prop          | Description                                                            |
| ------------- | ---------------------------------------------------------------------- |
| `as`          | [Changes HTML element](#change-html-element)                           |
| `pss`         | [Apply theme-aware styles](#theme-aware-styles)                        |
| `variant`     | [Set component style variant](#variants) (see docs for each component) |
| `colorScheme` | [Set color scheme](#color-scheme)                                      |

### Style props

| Prop | Description             |
| ---- | ----------------------- |
| `fg` | Text color (foreground) |
| `bg` | Background color        |
| `w`  | Width                   |
| `h`  | Height                  |
| `s`  | Width & height          |
| `m`  | Margin                  |
| `ml` | Left margin             |
| `mr` | Right margin            |
| `mt` | Top margin              |
| `mb` | Bottom margin           |
| `mx` | Left & right margin     |
| `my` | Top & bottom margin     |
| `p`  | Padding                 |
| `pl` | Left padding            |
| `pr` | Right padding           |
| `pt` | Top padding             |
| `pb` | Bottom padding          |
| `px` | Left & right padding    |
| `py` | Top & bottom padding    |

### Theme

| Key              | Description                                    |
| ---------------- | ---------------------------------------------- |
| `colors`         | text, background, caret, border, outline color |
| `sizes`          | width & height (including min/max), flex-basis |
| `space`          | margin, padding, gap, left, right, top, bottom |
| `borders`        | shorthand borders for each direction           |
| `borderWidths`   | border widths for each direction               |
| `borderStyles`   | border styles for each direction               |
| `fonts`          | custom font families                           |
| `fontSizes`      | custom font sizes                              |
| `fontWeights`    | custom font wights                             |
| `lineHeights`    | custom line heights                            |
| `letterSpacings` | custom letter spacings                         |
| `radii`          | border radii for each direction                |
| `shadows`        | box and text shadows                           |
| `transitions`    | shorthand transitions                          |
| `animations`     | shorthand animations                           |
| `durations`      | durations for animation and transition         |
| `timings`        | timing functions for animation and transition  |

## Examples

### Change HTML element

```jsx live
<Box as="main">Main</Box>
<Box as="a" href='https://google.com' target='_blank'>Link</Box>
```

### Theme-aware styles

Use `pss` prop to add styles using values defined in your [theme](#theme).

```jsx live
<Provider
  theme={{
    breakpoints: { $: null, $md: '(min-width: 600px)' },
    space: [0, 4, 8, 16, 32],
    colors: { primary: 'blue', foreground: 'white' },
    fonts: { primary: 'sans-serif' },
  }}
>
  <Box
    pss={{
      color: 'foreground',
      backgroundColor: 'primary',
      fontFamily: 'primary',
      width: 128,
      height: 64,
      padding: 2,
      marginX: 'auto',
      $md: {
        width: 256,
        height: 78,
      },
    }}
  >
    Box
  </Box>
</Provider>
```

### Variants

Use `variant` prop to set styles using values from your [theme](#theme).

```jsx live
<Provider
  theme={{
    boxes: {
      circle: {
        display: 'grid',
        placeItems: 'center',
        color: 'white',
        backgroundColor: 'red',
        borderRadius: 64,
        size: 128,
        fontFamily: 'sans-serif',
      },
    },
  }}
>
  <Box variant="circle">Box</Box>
</Provider>
```

### Color Scheme

Use `colorScheme` prop to choose different set of colors (powered by CSS variables). All elements inside will use these variables, following regular CSS inheritance rules.

```jsx live
<Provider
  theme={{
    colorSchemes: {
      light: {
        '--colors-foreground': '#222222',
        '--colors-background': '#eeeeee',
      },
      dark: {
        '--colors-foreground': '#eeeeee',
        '--colors-background': '#222222',
      },
    },
  }}
>
  <Box bg="background" fg="foreground" s={128} colorScheme="dark">
    Box
  </Box>
  <Box bg="background" fg="foreground" s={128} colorScheme="light">
    Box
  </Box>
</Provider>
```

### Responsive styles

```jsx live
<Provider
  theme={{
    breakpoints: {
      $: null,
      $md: '(min-width: 900px)',
    },
  }}
>
  <Box
    pss={{
      size: 64,
      backgroundColor: 'primary',
      $md: {
        size: 128,
        backgroundColor: 'accent',
      },
      ':hover': {
        backgroundColor: 'lime',
      },
    }}
  />
</Provider>
```

```jsx live
<Provider
  theme={{
    breakpoints: {
      $: null,
      $md: '(min-width: 900px)',
    },
  }}
>
  <Box
    s={{ $: 64, $md: 128 }}
    bg={{ $: 'primary', $md: 'accent', ':hover': 'lime' }}
  />
</Provider>
```

## Sizing

```jsx live
<Box w={64} h={28} bg="foreground" />
```

```jsx live
<Box s={32} bg="foreground" />
```

```jsx live
<Provider
  theme={{
    sizes: {
      foo: 64,
      bar: 128,
    },
  }}
>
  <Box w="foo" h="bar" bg="foreground" />
</Provider>
```

```jsx live
<Provider
  theme={{
    sizes: {
      bar: 128,
    },
  }}
>
  <Box
    pss={{
      size: 'bar',
      backgroundColor: 'foreground',
    }}
  />
</Provider>
```
