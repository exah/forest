# ZStack

```js
import { ZStack } from 'piny-forest'
```

Based on [Grid](#grid). Items placed one after another.
[Idea from Swift UI.](https://developer.apple.com/documentation/swiftui/zstack)

```jsx live
<ZStack>
  {['red', 'orange', 'yellow', 'green', 'blue'].map((color, index) => (
    <Box key={color} s={100} bg={color} mt={index * 10} ml={index * 10} />
  ))}
</ZStack>
```

```jsx live
<ZStack placeItems="center" fg="white" bg="lime">
  <Image
    src="https://placekitten.com/1600/800"
    width={800}
    height={400}
    alt="Cute kitten"
    pss={{ height: 'auto', maxWidth: '100%' }}
  />
  <Box bg="red" s={100} />
  <Box
    bg="yellow"
    pss={{
      placeSelf: 'end start',
      margin: 's.24',
      size: 40,
    }}
  />
</ZStack>
```
