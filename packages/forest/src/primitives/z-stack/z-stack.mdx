# ZStack

```js
import { ZStack } from 'piny-forest'
```

Based on [Flex](#flex). Items positioned one after another on z-axis.
[Idea from Swift UI.](https://developer.apple.com/documentation/swiftui/zstack)

  <Box bg="red" s={50} />
  <Box bg="yellow" s={75} pss={{ left: 's.16', bottom: 's.16' }} />
