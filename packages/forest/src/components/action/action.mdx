# Action

```js
import { Action } from 'piny-forest'
```

Small text-like button component. Accepts same variants as [Text](#text),
but behaves like [Button](#button).

```jsx live
<Action variant="primary">Primary</Action>
<Action variant="tertiary">Tertiary</Action>
<Action variant="secondary">Secondary (default)</Action>
<Action variant="accent">Accent</Action>
<Action variant="negative">Destructive</Action>
```
